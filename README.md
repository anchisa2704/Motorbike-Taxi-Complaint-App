## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/anchisa2704/Motorbike-Taxi-Complaint-App.git
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the Expo development server:

    ```bash
    npm start or npx expo start
    ```
